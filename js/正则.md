## 1、不允许存在空格和中文

```js
存在返回true
/[^\w\.\/]/ig.test('45 6')
true
```

## 2、不允许中文空格数字

```
!/^[A-Za-z_]+$/g
```

## 3、正则匹配ip地址

```js
(/^((2[0-4]\d|25[0-5]|[01]?\d\d?)\.){3}(2[0-4]\d|25[0-5]|[01]?\d\d?)$/).test('192.168.109.10')
true
```

## 4、匹配url

```js
//匹配数字、字母、下划线
/\/api\/v1\/cloud\/\w+\/detail/
```

## 5、正则密码

```js
//至少8位，并且至少含有1字母1数字1特殊字符
^\S*(?=\S{8,})(?=\S*\d)[A-Za-z\d](?=\S*[!@#$%^&*? ])\S*$
```
